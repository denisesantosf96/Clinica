# 🏥 Sistema de Clínica

## 📌 Descrição
Sistema web desenvolvido para gerenciamento de clínicas, permitindo o controle de pacientes, médicos e consultas.

O objetivo é facilitar a organização dos atendimentos e centralizar as informações.

## 🚀 Tecnologias
- C#
- ASP.NET Core
- SQL Server / PostgreSQL
- Entity Framework Core
- HTML, CSS, Bootstrap

## ⚙️ Como rodar o projeto
1. Clonar o repositório
2. Abrir no Visual Studio
3. Configurar a string de conexão com o banco
4. Executar o projeto

## 🎯 Funcionalidades
- Cadastro de pacientes
- Cadastro de médicos
- Agendamento de consultas
- Edição e exclusão de registros
- Validações de dados
